# CoursBasesUnityGD2\_RollABall

Cours de début d'année avec Gilbert sur Unity pour nous en apprendre les bases





Objectif : Un shoot'em up d'une sphere qui tue des ennemies dans une chambre d'enfant effet cartoon. la sphere est equiper d'une tourelle monter qui vise a l'endroit de la sourit et qui peut etre améliorer avec plein d'arme. Le joueur pourrais aussi etre amélioré. Un jeu du type Vampire Survivor ou le joueur devrais survivre a plusieurs types d'ennemi et ou a chaque manche le joueur pourra s'ameliorer.
